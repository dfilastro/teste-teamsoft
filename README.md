# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/dfilastro/teste-teamsoft.git
```

```bash
# Install Dependencies
$ yarn
```

```bash
$ yarn start
```

Go to http://localhost:3000/ to see the result.
